Description

This application is designed to perform Principal Component Analysis (PCA). PCA is a computational method for reducing the complexity of high-dimensional data by projecting the data onto a lower-dimensional space. Moreover, it is an explorative method to gain insight into the dependencies of the data. By uploading your dataset and selecting your variables of interest, you will receive a PDF report of the PCA with additional explanations and visualizations.

Data Input and Processing:
The data is read using the fread function from R, which receives a CSV file as input. The separator and header are detected automatically and do not need to be specified manually. For information on how to create your CSV file, click here.
